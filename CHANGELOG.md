# Changelog

## [1.0.0] - 2026-05-18

### Added
- First marketplace release.
- Full creation flow: persistent agent with identity, ishiki, optional public-ishiki.
- Overlay mode: merge short personality additions on top of Hanako's default templates.
- Replace mode: fully custom identity/ishiki files.
- Avatar support: `avatarUrl` (remote download) and `avatarDataUrl` (inline data URL).
- Primary-agent-only enforcement with rollback on failure.
- Comprehensive smoke test suite (tool exports, full creation, replace mode, validation, permissions).
