# OpenCode Providers

接入 [OpenCode](https://opencode.ai) 的模型订阅服务：

## OpenCode Zen

按量付费模式，覆盖 GPT 5.5、Claude Opus 4.7、Gemini 3.1 Pro、Kimi K2.5、Qwen3.6 Plus 等全系列主流模型。

- **Base URL**: `https://opencode.ai/zen/v1`
- **认证**: API Key（从 [opencode.ai/auth](https://opencode.ai/auth) 获取）
- **协议**: OpenAI 兼容

## OpenCode Go

订阅制模式（首月 $5，之后 $10/月），覆盖 DeepSeek V4 Pro/Flash、Kimi K2.5/K2.6、Qwen3.5/3.6 Plus、MiniMax M2.7 等精选开源模型。

- **Base URL**: `https://opencode.ai/zen/go/v1`
- **认证**: API Key（先订阅 Go 后从 [opencode.ai/auth](https://opencode.ai/auth) 获取）
- **协议**: OpenAI 兼容

## 使用方式

1. 安装此插件后，进入设置 → 供应商
2. 在 API Key 列表中找到 **OpenCode Zen** 或 **OpenCode Go**
3. 填入你的 API Key，保存即可
4. 在助手设置中选择对应模型
