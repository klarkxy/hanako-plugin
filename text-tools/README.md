# 文本工具 (text-tools)

> 文本处理工具集，为 Hanako Agent 提供字数统计等文本分析能力。

## 功能

| 工具 | 说明 |
|------|------|
| `count_chars` | 统计文档的中文字数、所有字符数、UTF-8 字节数 |

## count_chars 工具

统计指定文档的字数与字符数，支持两种输入方式：

- **filePath**：传入文档的绝对路径，工具读取文件后统计
- **text**：直接传入文本内容进行统计

### 返回统计项

| 项目 | 说明 |
|------|------|
| 中文字数 | 使用 Unicode `Script=Han` 属性识别所有 CJK 汉字（含基本区、扩展区、兼容区） |
| 非中文其他字符 | 所有字符减去中文字符数 |
| 所有字符总数 | `String.length`，含字母、数字、标点、空格、换行等 |
| UTF-8 字节数 | 文本编码后的字节大小 |

## 使用示例

Agent 调用示例：

```
请用 count_chars 统计文件 d:/notes/chapter1.md 的字数
```

或直接统计文本：

```
请统计下面这段文字的中文字数和总字符数：……
```
